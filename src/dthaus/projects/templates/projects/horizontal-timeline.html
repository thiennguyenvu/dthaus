Create Project

You need fill all require fields below:
  • Project Name
  • Due Date
  • Customer. Create new customer here
  • Project Manager

Phase Settings

  • You can create a name of phase and inherit this name for next project
  • Select/Unselect phases you want add to your project
  • Save your choices with 'Save Phase Settings' before click 'Add Phase to Project'

Phase Edit

  • You can change phase name and it doesn't affect with phase list of step Phase Settings
  • Add more phase
  • Update all changes
  • View task details in each phase

Task View & Task Settings

  • You can add more Task and create a name of Task, inherit this name for next phase
  • Select/Unselect task you want add to your phase
  • Save your choices with 'Save Task Settings' before click 'Add Task to Phase'

Task Edit

  • Fill information of Task and click 'Update Task'
